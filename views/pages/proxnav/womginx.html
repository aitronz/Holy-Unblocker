Incredibly fast proxy using only NGINX as the backend server.

Classic Stealth

More Information:

Has decent support for a large amount of sites, particularly for Discord and reCAPTCHA.
Some limitations would be that with some filters, pages may still be blocked.
In the case of issues, be sure to wait for the page to fully load before navgating to the site you are on.
Please read the FAQ page regarding issues with reCAPTCHA on Womginx.
